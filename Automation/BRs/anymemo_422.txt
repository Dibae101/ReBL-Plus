Bug Report Title:
Application crash after rotating screen

Bug Report Issue:
The application crashes when the screen is rotated during the learning process.

### Steps to reproduce:
1. Auto-rotate feature is enabled in system
2. Open AnyMemo
3. Start learning process in AnyMemo
4. When question is visible on screen, rotate the device
5. Error message appears: "Unfortunately, AnyMemo has stopped."
6. App crashes

### Versions:
* AnyMemo 10.9.922 (F-Droid build)
* CyanogenMod 13.0-20161219-NIGHTLY-m7

### Expected behavior:
The app should handle screen rotation gracefully and maintain the current state during the learning process.

### Actual behavior:
App crashes immediately upon screen rotation while displaying a question during the learning process.

Comments:
Issue was reported and fixed on January 5, 2017.
Bug affects version 10.9.922.
